3. Or forward the eLaw email to yourself and copy the email text
4. Paste it into the box below (Ctrl+V)
5. Review the parsed info and click Import Client

📁 OneDrive Integration
Each client profile has a Discovery Files section where you can paste their OneDrive folder link.

To link a folder:
1. Open a client's profile
2. Scroll to 📁 Discovery Files
3. Click ✎ Edit Link
4. Paste the shared OneDrive folder URL

To get a folder link in OneDrive:
Right-click the folder → Share → Copy link
